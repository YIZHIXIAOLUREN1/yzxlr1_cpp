### **坐标与朝向约定**

| 约定 | 定义 |
| :--- | :--- |
| **中轴线** | X = 0, Z = 0，铸剑巨剑的轴线，垂直贯穿整个结构 |
| **朝向** | 面向负Z方向为“正面”（即站在正Z侧，面朝负Z方向观看结构正面） |

| **相对高度** | 文档中所有“h=”标注均为相对于结构最低点（发射井底部 Y = 46）的高度，即 h = Y − 46 |

## **第二部分：地面层（h 0–25 / Y 46–71）**

### **2.1 发射井 (The Launch Pit)**
**位置**：中心 (0, 0)，从地面 h=18 向下挖掘至 h=0。

**尺寸**：
*   **井口**：直径 16 格（半径 8），正八边形近似圆
*   **井底**：直径 12 格（半径 6），略收窄
*   **深度**：18 格

**分层建造**：
```
h=18（地面层，井口）：
    正八边形开口，外径16格。
    边框：黑石砖，宽2格。
    边框顶面：嵌入铁活板门（打开状态），模拟金属护栏。

h=17–12（上部井壁）：
    材质：深板岩砖为主体，每隔3层嵌入一圈深板岩瓦作为水平分隔线。
    四个正交方向（±X, ±Z）各嵌入一条竖直的红石灯带（宽1格），
    模拟熵监测导轨。红石灯保持ON状态。
    内径从16格渐缩至14格（每3层内收1格）。

h=11–4（下部井壁）：
    材质过渡为深板岩+深板岩圆石混合（表现粗糙的熵侵蚀质感）。
    红石灯带继续，但每隔2格断开1格（模拟损坏/烧蚀）。
    内径继续渐缩至12格。
    从h=8开始，壁面上随机嵌入岩浆块（占壁面面积约15%），
    模拟地脉热量上涌。

h=3–1（井底过渡）：
    壁面改为黑曜石+深板岩圆石混用。
    岩浆块密度增加至30%。
    在四角各放置一个灵魂灯笼（蓝色火焰，暗示以太残留）。

h=0（井底）：
    中央 6×6 格铺满岩浆块，上覆一层熔岩（源方块）。
    四周 1 格宽的环形台阶用黑曜石铺设。
    正中央的3×3区域放置红色荧石（用红色玻璃覆盖荧石模拟），
```

### **2.2 地面熵疤区域 (Entropy Scar Zone)**
**范围**：以 (0,0) 为中心，半径 100 格的圆形区域，h = 18（地面层）。

**分区设计**：

| 径向距离 | 地表处理 | 裂缝密度 | 特殊元素 |
| :--- | :--- | :--- | :--- |
| **0–8格（井口边缘）** | 黑曜石 + 黑色混凝土，100%硬化 | — | 发射井护栏 |
| **8–25格（重度熵疤）** | 黑色混凝土为主(70%)，灰色混凝土(20%)，黑曜石(10%) | 高密度裂缝网（每4–6格一条） | 裂缝内嵌岩浆块；地面偶尔嵌入粗金块碎片；紫色地毯苔藓团 |
| **25–50格（中度熵疤）** | 灰色混凝土(50%)，黑色混凝土(30%)，泥土(20%) | 中密度裂缝（每8–12格一条） | 裂缝内嵌红石灯（模拟暗红微光）；零星紫色地毯+缠怨藤团 |

**裂缝建造方法**：
1. 在地面层沿随机方向挖出1格宽、1–3格深的沟槽。
2. 沟槽底部放置岩浆块或红石灯（ON）。
3. 沟槽上方可选盖一层铁栏杆（营造格栅感）或留空。
4. 主要裂缝从井口呈放射状向外延伸，次要裂缝在其间做不规则分叉。

### **2.3 悬浮碎石场 (Floating Debris Field)**
**范围**：井口周围半径 80 格内，h = 19–40 (Y = 65–86) 之间散布。

**碎石规格**：

| 类型 | 数量（建议） | 尺寸 | 悬浮高度(h) | 材质 |
| :--- | :--- | :--- | :--- | :--- |
| **微型碎石** | 40–60块 | 1×1×1 至 2×2×1 | h 19–25 | 圆石、安山岩 |
| **小型碎石** | 20–30块 | 3×2×2 至 4×3×2 | h 20–30 | 石头+圆石混合，底面附泥土 |
| **中型岩块** | 8–12块 | 5×4×3 至 7×5×4 | h 22–35 | 石头为主，安山岩为辅，可嵌少量矿石装饰 |
| **大型巨石** | 2–4块 | 8×6×5 至 10×8×6 | h 28–40 | 石头+闪长岩+安山岩不规则混用，底面有“根系”般的泥土悬垂 |

**布局原则**：
*   碎石密度从井口向外递减——最密集区域在半径 15–40 格之间。
*   碎石不要排列整齐——随机旋转、随机间距、随机高度。
*   大型巨石建议放置在半径 20–35 格范围，高度 h 28–40 之间。
*   所有碎石均为悬浮态（无支撑柱），利用 Minecraft 方块不需物理支撑的特性。
*   可在个别大型巨石底面挂 1–2 根缠怨藤（紫色变体），暗示以太连接。

### **2.4 地脉冷却管道 (Geothermal Cooling Pipes) ×3**
三根管道从地面基座向地下三个不同方向倾斜延伸。地面入口位于距中心约 30–40 格处。

| 管道编号 | 地面入口位置 | 倾斜方向 | 地面可见段长度 | 倾角（近似） |
| :--- | :--- | :--- | :--- | :--- |
| **管道 A** | (−30, +20) | 向西南下倾 | 地面以上 8 格 + 地下 15 格 | ~30° |
| **管道 B** | (+25, +30) | 向东南下倾 | 地面以上 10 格 + 地下 18 格 | ~40° |
| **管道 C** | (+5, −35) | 向北下倾 | 地面以上 6 格 + 地下 12 格 | ~20° |

**单根管道横截面（5×5格）**：
```
    C C C C C          C = 青色玻璃（管壁）
    C B B O C          B = 蓝冰（冷液上行通道，中心）
    C B B O C          O = 橙色玻璃（热废液下行通道，外环）
    C O O O C          内部空间3×3格
    C C C C C
```
**建造细节**：
1.  管壁使用青色染色玻璃，透明可见内部双色流体。

### **3.5 剑刃 (Blade Edge) 处理**
在 Minecraft 中无法做出真正的锋利刃缘，但可用以下技巧暗示锐利度：

*   **h 45–155（主剑身）**：在截面最外侧的两列方块使用台阶或半砖削薄：

### **3.6 “以太呼吸”表现**
设计文档描述剑身表面金色纹路每约2.4秒微微增亮然后回暗。在 Minecraft 中可通过以下方式近似表现：
*   在五角纹路图案的交汇节点处嵌入红石灯（替代部分荧石），连接一个周期约2.4秒的红石脉冲发生器（中继器链路或漏斗时钟）。
*   红石灯亮/灭的交替会使纹路节点在金光和暗金之间切换。
*   每面剑身上约6–8个节点同步控制即可产生整体“呼吸”效果。
*   **注意**：此为可选进阶功能。红石线路需隐藏在血槽内部或剑身厚度之内。

### **5.6 对角线方向注入器的建造技巧**
正交方向（±X， ±Z）的注入器沿坐标轴直线延伸，建造简单。对角线方向（45°， 135°， 225°， 315°）的注入器需要在 X-Z 平面上以 45° 角前进：

**方法**：每前进1格 X 同时前进1格 Z（即沿对角线）。截面保持不变，但整体形状呈“阶梯锯齿”。

**减轻锯齿感的技巧**：
1.  使用台阶/半砖在阶梯的内角和外角进行平滑
2.  将剑身截面从3格宽改为4格宽（在对角线方向多出1格有助于平滑视觉效果）
3.  在阶梯转折处放置玻璃板或末地烛作为视觉过渡

### **5.7 注入器剑尖与约束环的嵌合**
每把注入器的最后2–3格嵌入约束环外壁。具体做法：
1.  在约束环外壁的对应冲击点位置，挖出一个 3×3×3 格的凹槽
2.  注入器的最后3格插入凹槽
3.  凹槽内壁改为白色混凝土
4.  注入器尖端方块改为白色混凝土（与冲击点融合）
5.  凹槽周围1格范围内放置白色玻璃板和末地烛，模拟冲击处的白色光幕

### **6.2 螺旋几何**
双螺旋结构由两条对称的螺旋臂组成，相位差180°。每条螺旋臂宽2–3格，沿着中心轴旋转上升。

**中心轴直径随高度变化**：
```
底部 h=168：半径 4 格
h=215：      半径 5 格
h=270：      半径 7 格
h=300：      半径 9 格
顶部 h=310：半径 11 格（漏斗口）
```

**建造方法——螺旋臂的逐层搭建**：
1.  先用脚手架在中心轴位置建一根临时立柱（从 h=168 到 h=310，1×1格）。
2.  在每个高度 h 上，按当前半径 r(h) 画一个圆。
3.  计算当前旋转角度 θ(h) = θ₀ + Σ(Δθ per level)。
4.  在圆周上 θ 位置放置螺旋臂 A 的方块（弧长约2–3格），在 θ+180° 位置放置螺旋臂 B 的方块。
5.  完成后拆除中心立柱。

螺旋臂之间的空间不应完全为空——需要用玻璃部分填充以表现以太流的粘稠度：
*   **下段（h 168–215）**：螺旋臂之间每隔1–2格放置白色玻璃块，形成“稠密白雾”，但保持足够空隙以透光。

*   **上段（h 270–310）**：填充密度降低至30%——这里以太更加稀薄、更接近气态，使用交替的橙色和紫色玻璃块，它们之间留有大量空隙，光线可以穿透。

### **6.5 漏斗口 (The Funnel Mouth) — h 300–318**
螺旋光柱在 h=300 处开始急剧扩张，形成漏斗状开口。漏斗口从直径18格（h=300）扩展至直径约40格（h=318），然后与法阵膜面的底面衔接。

**漏斗壁**：

*   漏斗壁厚2格——外层为染色玻璃，内层为对应颜色的混凝土（增强结构感）。
*   漏斗口边缘（h=318）使用白色混凝土+海晶灯镶边，标记从螺旋向法阵的过渡。

### 三层结构详解

**上表面（h = 327 或 h = 326，即膜面最高层）——铭文功能层**：

| 元素 | 材质 | 位置 | 功能含义 |
|------|------|------|---------|
| 网格线 | 品红色混凝土 | 六边形边线，宽1格 | 流导符的投射——引导 Logos 秩序流从计算核心向法阵中心输送 |
| 六边形内底 | 紫色混凝土 | 网格线围成的区域 | 锚定符的承载面——固定法阵膜面的局部 Ω 值 |
| 中心铭文 | 紫水晶簇（朝上放置） | 每个六边形的几何中心，1格 | 锚定符的投射节点——每个簇代表一个独立的 Ω_target 定义点 |
| 节点连线 | 紫水晶芽（小型/中型，贴壁放置） | 沿网格线每隔 2–3 格放置 1 个，贴在品红混凝土侧面 | 流导符的方向性指示——暗示秩序场的流动方向 |

**建造顺序建议**：
1. 先铺满整个右侧区域的紫色混凝土底面（h = 325 层）
2. 在 h = 326 层用品红色混凝土画出六边形网格线
3. 在每个六边形中心放置紫水晶簇
4. 在网格交叉顶点放置紫水晶块
5. 沿网格线侧面贴紫水晶芽

**中间层（h = 325）——结构层**：

- 全部使用紫色混凝土实心填充。
- 在与中央过渡带相邻的 8 格范围内，混入约 30% 的品红色混凝土（颜色从纯紫向中心过渡带的白色缓慢渐变）。
- 每隔 15–20 格沿 X 方向嵌入一条 1 格宽的海晶灯带（沿 Z 方向延伸全宽），作为结构内部的"秩序总线"发光指示。

**下表面（h = 324）——光渗透层**：

这一层是从地面仰望时可见的面——需要让上方铭文层的光效部分透过来。

- 主体：紫色玻璃（约 60%）+ 紫色混凝土（约 40%）交替铺设。
- 紫色玻璃的位置对应上方的六边形内部区域——从下方仰望时，可以通过玻璃隐约看到上方紫水晶簇的辉光透下来，形成规则的紫色光点阵列。
- 紫色混凝土的位置对应上方的网格线——网格线区域不透光，从下方看呈现为暗色的网格骨架。
- 在海晶灯带的正下方位置（h=324），使用海晶灯代替紫色玻璃，使结构内部的"秩序总线"光带同时从底面可见——从地面看去，是几条水平贯穿的明亮光线镶嵌在紫色蜂巢底面中。

### 边缘处理

**外侧边缘（距中心 95–105 格处）**：

- 六边形网格在边缘处不做完整裁切——允许最外圈的六边形被膜面边缘截断，形成不完整的半六边形和三角形残片。
- 边缘最后 3–5 格范围内，材质从紫色混凝土渐变为紫色玻璃→紫色玻璃板（竖直放置，仅存骨架），制造一种"秩序场正在衰减消散"的边界效果。
- 最外缘放置竖直的紫色玻璃板（每隔 2 格一片），如同一排正在溶解的紫色栅栏——秩序的边界。
- 边缘处每隔 10–15 格保留一个完整的紫水晶块交叉节点（孤立地悬浮在边缘外1–2格处），暗示秩序场的惯性延伸——即使膜面物理边界已到，Logos 场仍然向外衰减了几格才彻底消失。

**与中央过渡带的交界（X ≈ +5 至 +12 附近）**：

- 从纯紫色混凝土底面逐渐混入白色混凝土：
  - X = +12 处：100% 紫色
  - X = +10 处：80% 紫色 + 20% 白色（随机混合）
  - X = +8 处：50% 紫色 + 50% 白色
  - X = +5 处：20% 紫色 + 80% 白色，融入过渡带
- 六边形网格在过渡区逐渐"失焦"：
  - X = +12 至 +10：网格线完整，品红色混凝土
  - X = +10 至 +8：网格线开始断续——每条边有 1–2 格缺口（品红混凝土被白色混凝土替换）
  - X = +8 至 +5：网格线大部分消失，仅存零星的品红色方块残片和孤立的紫水晶簇——如同秩序的几何蓝图正在被混沌侵蚀溶解
- 紫水晶簇在过渡区的处理：
  - X > +10：每个六边形中心标准放置，朝上
  - X = +8 至 +10：部分紫水晶簇改为侧面放置（朝向中心/+X方向倾斜），暗示秩序场正在被拉向过渡带
  - X = +5 至 +8：仅残余 2–3 个紫水晶芽（小型），贴在白色混凝土表面，几乎不可见

### 内嵌细节与视觉节奏

**"秩序脉动"光带**：

在六边形网格中，选取 3–4 条从浑天仪方向（+Z）向法阵中心方向（−Z/原点）延伸的网格线路径作为"主秩序流通道"。这些通道上的网格线材质从品红色混凝土升级为海晶灯——形成 3–4 条明亮的发光线条横贯整个右侧膜面，从后方（+Z边缘附近，浑天仪一侧）一直延伸到中央过渡带附近。

- 每条主通道宽度仍为 1 格，但每隔 8 格在通道两侧各添加 1 格海晶灯，形成短暂的"加宽闪光节"。
- 这些主通道的视觉意义：代表计算核心向法阵中心持续输送 Logos 计算结果的数据高速公路。

**偶发性"计算异常"斑块**：

为避免右侧过于完美单调，在 180–220 个六边形中选取 3–5 个作为"计算异常"单元——这些六边形内部的紫色混凝土被替换为：

- 1 个：品红色混凝土（过度活跃的计算节点，颜色偏亮）
- 1 个：灰色混凝土（失活的计算节点，颜色灰暗，中心紫水晶簇改为紫水晶芽/小型，暗示功率不足）
- 1 个：中心紫水晶簇改为末地烛（白色光芒替代紫色，暗示该节点正在执行特殊指令——可能是剑身微调计算的高优先级任务）
- 1–2 个：六边形内部不放紫色混凝土，改为紫色玻璃（完全透明，从上方可以直接看穿到底面的天空/下层结构），暗示该区域的以太膜面极其稀薄——秩序场在此处接近"撕裂"的临界状态

这些异常斑块应随机分布（不要形成规律），每个之间至少间隔 20 格以上，避免给人"大面积损坏"的错觉——它们是偶发的、受控的异常，不是系统性故障。

### 右侧与浑天仪的视觉关联

浑天仪位于法阵膜面的 +Z 方向（后方），其数据瀑布从底部向下延伸。右侧六边形网格与浑天仪之间需要建立视觉上的"连接感"：

1. **主秩序通道的起点**：3–4 条主秩序光带从膜面后缘（+Z方向，面向浑天仪的一侧）起始。在膜面后缘的边缘处，这些光带的起始端做一个 2×2 格的海晶灯节点（比正常网格线粗），节点上方各放 1 根末地烛（向上指向浑天仪方向）。
2. **数据瀑布的接地点**：浑天仪最长的几条数据瀑布（粗流）的末端，应指向法阵膜面右侧后部的这些起始节点——虽然瀑布末端与膜面之间可能有 10–30 格的高度差（瀑布在 h ≈ 280 结束，膜面在 h ≈ 325），但从视觉上让它们在空间中大致对应，传达"数据从计算核心流下，被法阵右侧的秩序网格接收并向中心输送"的信息。
3. **可选连接桥**：在瀑布末端与膜面起始节点之间架设 2–3 条品红色玻璃板构成的斜向"数据桥"（每条由 5–10 片玻璃板组成的倾斜序列），物理性地连接两个结构体。

### 7.3 法阵膜面的边缘处理

法阵不是一个完整的圆盘——其边缘应呈现参差不齐的破损形态：

- **右侧（紫色区域）边缘**：相对整齐，但每隔 15–20 格有一处 3–5 格深的凹缺（表现为熵侵蚀）。边缘用紫色玻璃板竖直放置，创造半透明的"能量膜"边界感。
- **左侧（橙红区域）边缘**：极度不规则。大量 5–15 格长的悬臂尖刺从边缘向外伸出，部分尖刺末端已经断裂（仅留 1–2 格的残根）。边缘没有明确的分界线——橙色混凝土和空气之间是锯齿状的犬牙交错。
- **整体轮廓**：从俯视看，法阵膜面的轮廓像一片被部分烧蚀的巨大叶片——右半规整如切割，左半毛躁如撕裂。

### 7.4 法阵膜面的底面处理

膜面的底面（朝下的面）同样需要处理，因为从地面仰望时这是主要可见面：

- **右侧底面**：紫色混凝土平面，六边形网格线的荧光可以从上方透过（使用玻璃方块替换部分混凝土，使顶面的紫水晶簇光线可以泄漏到底面）。
- **左侧底面**：不规则，大量悬垂的橙色玻璃"钟乳石"和熔岩滴状物。使用橙色玻璃板、指向下方的末地烛、悬挂的锁链等营造"火焰天花板"的效果。
- **中央过渡带底面**：白色玻璃，从下方可以看到海晶灯的光晕透过来。

### 7.5 漏斗衔接与膜面中央下凹

法阵膜面在中央处向下凹陷，与螺旋光柱的漏斗口衔接：

- 膜面在距中心 20 格处开始下弯，到中心处下凹至 h=310（漏斗口顶部）。
- 下凹区域的材质渐变为白色混凝土→白色玻璃→品红色玻璃（向漏斗内过渡）。
- 下凹区域的边缘（直径 40 格的圆形口）用荧石+白色混凝土镶嵌一圈发光的边框，标记法阵与螺旋光柱的分界线。

### 8.2 火球核心 (The Fireball)

**球体建造**：直径 26 格的近似球体。使用标准 Minecraft 球体建造图表（26 直径 = 半径 13）。

**表面材质——对流格元模拟**：

火球表面不是均匀的橙色——需要模拟太阳表面的米粒组织（对流格元）：

- **格元中心（亮区）**：橙色混凝土 + 黄色混凝土混用，中心偶尔嵌入荧石（约每个格元 1–2 个）。

- **格元尺寸**：每个格元在球面上约 5–7 格宽（不规则的多边形）。整个球面约被 30–50 个格元覆盖。

**建造步骤**：

1. 先用橙色混凝土搭建完整的 26 格直径球体。

4. 在每个亮格元中心嵌入 1 个荧石方块。

**暗斑 (Dark Spots)**：

每颗火球表面需要建造 3 个暗斑（紫色锁链的锚定点）：

- 尺寸：每个暗斑直径 3–5 格的圆形/椭圆形区域。
- 材质：棕色混凝土为底，灰色陶瓦为边缘过渡。暗斑中心无发光方块。
- 位置：分布在火球背面（远离法阵中心的一侧），朝向机械爪方向。三个暗斑大致呈三角形分布，间距 8–10 格。
- 每个暗斑的精确中心处放置 1 根锁链方块（Chain），作为紫色锁链的接入锚点。

### 8.3 日冕弧 (Coronal Arcs)

  - 中段：橙色玻璃（主体）+ 黄色玻璃（内芯，发光用）

**弧线建造方法**：

### 9.2 三层浑天仪环的建造

- **环B（经线环1）**：竖直放置，沿 X 轴方向。同材质同截面。
- **环C（经线环2）**：竖直放置，与环B夹角约 60°（不是 90°，故意制造不对称以增加视觉复杂度）。同材质同截面。

每条环的建造方法：
1. 使用 Minecraft 圆形图表生成半径 18 格的圆。
2. 在圆周上每个位置放置 3×3 截面的方块组（紫珀块为主体，中心 1 格用紫珀柱）。
3. 环面上每隔 6–8 格嵌入一个紫水晶簇（铭文），紫水晶簇朝外放置。
4. 三条环交叉处（共 6 个交点）做特殊处理：交点用紫水晶块（3×3×3 格的节点球）标记，每个节点球上放 4 根末地烛（四个方向各一根），模拟交叉点的紫色闪光。

**中层（直径 22 格）——2 条大圆环：**

- **环D（倾斜赤道环）**：倾斜约 30° 放置。材质：紫珀块+紫水晶块交替，环截面 2×2 格。
- **环E（倾斜经线环）**：倾斜方向与环D垂直。同材质同截面。
- 两环交叉处（4 个交点）各放置一个末地烛簇（4 根末地烛）。
- 中层环面上铭文密度更高：每隔 4 格一个紫水晶簇。

**内层（直径 14 格）——1 条大圆环 + 核心体：**

- **环F（核心环）**：倾斜约 45° 放置。材质：紫水晶块为主体，环截面 2×2 格。每隔 3 格一个紫水晶簇。
- **核心体**：在内层中心放置一个 4×4×4 格的实心方块体，材质为紫水晶块（外壁）+ 荧石（内核，仅中心 2×2×2 格）。这是浑天仪的"CPU核心"。
- **可选**：在核心体位置放置一个信标（Beacon），激活紫色光束向上投射。信标需要金字塔底座，可将其隐藏在核心体内部（使用铁块/金块等建造 3×3 底座层）。

### 9.3 数据瀑布 (Data Waterfall)

从浑天仪底部向下悬垂的 Logos 场溢出可视化。

**结构**：一组长短不一的垂直悬垂柱，从浑天仪底部（h ≈ 314）向下延伸。

| 瀑布类型 | 数量 | 长度范围 | 截面 | 材质 | 颜色 |
|---------|------|---------|------|------|------|
| 粗流 | 3–4 条 | 30–45 格 | 2×2 格 | 紫色混凝土为壳，内嵌紫色玻璃 | 深紫色 |
| 细流 | 6–8 条 | 15–30 格 | 1×1 格 | 品红色玻璃 | 亮紫色 |
| 发丝流 | 10–15 条 | 5–15 格 | 紫色玻璃板（单片） | 紫色玻璃板 | 淡紫偏白 |

**布局**：瀑布从浑天仪底部的不同位置出发，向下垂直延伸。起点分布在浑天仪底部半球的各个位置（不集中在一个点）。最长的粗流可以延伸至 h ≈ 270（接近螺旋光柱上段），在末端用渐变的方式消散——最后 3–5 格从混凝土改为玻璃，再改为玻璃板，最终只剩 1 个末地烛。

**每条粗流的内部**：每隔 5 格嵌入 1 个末地烛（位于 2×2 截面的中心，替换 1 格紫色玻璃），使瀑布从内部发出柔和的白紫色光。

### 9.4 浑天仪黄金共振点闪光效果（可选）

设计文档描述每隔约 0.54 秒浑天仪会爆发紫色十二角星芒闪光。在 Minecraft 中可以通过以下方式近似：

- 在三层环体的 6 + 4 + 2 = 12 个交叉节点上分别放置红石灯。
- 用一个中央红石时钟（周期约 10 红石刻 ≈ 1 秒）控制所有红石灯同步闪烁。
- 红石线路沿着环体内部走线（隐藏在 3×3 或 2×2 截面的空心内部）。

## 第十部分：外围防御装甲板 (Outer Defense Armor Plates) — h 180–350 / 散布

### 10.1 总体布局

防御装甲板是一系列不连续的弧形板片，散布在主体结构周围，如同一条被打碎的外围防线。

- **分布高度**：主要集中在 h 200–340 范围内，在螺旋光柱和法阵膜面的外围飘浮。
- **分布半径**：距中轴线 85–120 格。
- **总数量**：15–22 片（含完整片和残缺片）。
- **弧度**：每片覆盖圆周的 12°–30°。

### 10.2 装甲板分类

| 类别 | 数量 | 弧长(格) | 高度(格) | 厚度(格) | 状态 |
|------|------|---------|---------|---------|------|
| 完整板 | 6–8 片 | 25–40 | 8–12 | 2–3 | 结构完好，仅有表面裂纹和边缘磨损 |
| 受损板 | 5–7 片 | 15–30 | 5–10 | 1–2 | 明显的融蚀痕迹，部分区域缺失，内部结构暴露 |
| 残片 | 4–7 片 | 5–15 | 3–6 | 1 | 仅存碎片，悬浮在原位。边缘严重烧蚀 |
| 幽灵锚点 | 3–5 个 | — | — | — | 装甲板完全蒸发后残留的悬浮锚定符残骸 |

### 10.3 完整板的建造

**主体结构**：

- 弧形板——沿以中轴线为中心、半径 90–110 格的圆弧弯曲。
- 材质：深板岩砖（外表面） + 深板岩瓦（内表面）。
- 板体内（2–3格厚度之间）嵌入功能层：外半层用深板岩砖，内半层用海晶石砖（代表衰减符铭文网络）。

**表面细节**：

- **外表面**（面朝外的面）：
  - 每隔 5–8 格刻一条竖直的泄压符纹路——用深板岩圆石替代深板岩砖，宽 1 格，形成深色条纹。
  - 纹路旁放置红色玻璃板（竖直，贴壁），模拟泄压符的暗红色闪烁。
  - 板面上随机分布 2–3 处哭泣的黑曜石方块——模拟熵灼伤的永久痕迹。

- **内表面**（面朝中轴线的面）：
  - 海晶石砖为底，每隔 4 格嵌入 1 个紫水晶簇（衰减符铭文的发光节点）。
  - 紫水晶簇发出微弱的紫色辉光，从内侧看去如同一面散布着紫色星辰的暗墙。

**边缘处理**：

- 板的上下边缘使用台阶/半砖削薄，创造从 3 格厚渐薄至 1 格的楔形边缘。
- 上边缘可略有不规则——用深板岩圆石替代部分台阶，制造轻微的磨损感。
- 下边缘更加不规则——参差 1–3 格的锯齿状轮廓，部分位置缺失 1–2 格（模拟历次铸剑的熵侵蚀累积）。

### 10.4 受损板的建造

在完整板的基础上进行"破坏"：

1. **切除 1–2 个不规则区域**（每个 3×3 至 5×5 格），形成贯穿板体的洞口。洞口边缘使用凝灰岩（Tuff）表现烧蚀面的粗糙质感。
2. **融蚀渐变**：从洞口边缘向外 2–3 格范围内，材质从深板岩砖→深板岩圆石→凝灰岩→红色玻璃渐变（从完好→损坏→烧蚀→发光）。
3. **内部结构暴露**：在洞口位置，板体内部的海晶石砖层可见。可在暴露的海晶石砖上放置单个紫水晶簇（已受损的铭文节点，光芒不完整）。
4. **板面裂纹**：用铁栏杆在板面上画出 2–3 条从洞口向边缘辐射的"裂纹线"。裂纹线下方（板体内部）嵌入红色玻璃（裂纹内的暗红微光残留）。

### 10.5 残片的建造

极小的碎片，仅 5–15 格长、3–6 格高、1 格厚。

- 材质：深板岩圆石为主，50% 以上面积被凝灰岩和红色玻璃覆盖（严重烧蚀）。
- 形状极不规则——没有直线边缘，全部用台阶和半砖制造锯齿轮廓。
- 每片残片上保留 0–1 个紫水晶簇（铭文残余，大部分铭文已因Ω退化而失效）。

### 10.6 幽灵锚点 (Ghost Anchor Points)

在装甲板已完全蒸发的位置上，空中仅存少量悬浮残骸：

- **结构**：3–5 个末地烛悬浮于空中，彼此间距 2–3 格，呈不规则簇状。中央放 1 个紫水晶块（暗金色锚定符残骸，在 Minecraft 中紫水晶块的淡紫色勉强近似暗金色意象）。
- **效果**：幽灵锚点周围 3 格范围内放置白色玻璃板（单片，无框），模拟该区域以太场的永久性轻微扰动。
- **位置**：在装甲板弧线的缺口位置上，处于本应存在装甲板的那段弧线上。

### 10.7 全息警告文字

在装甲板之间的较大缺口处（弧长 > 15 格的空隙），使用告示牌 (Signs) 和旗帜 (Banners) 构建全息文字效果。

**建造方法**：

1. 在缺口的中间位置，距中轴线约 95–105 格处，放置一根竖直的 2–3 格高的隐形支撑柱（使用屏障方块/Barrier Block，或在创造模式下使用结构空位方块）。如无法使用隐形方块，可用末地烛柱代替（1×1 格竖直排列），外观也可接受。
2. 在支撑结构的四面贴上告示牌。
3. 告示牌文字内容及颜色：

| 面向 | 文字内容 | 颜色 |
|------|---------|------|
| 面朝外 | `WARNING` / `⚠ ARMED ⚠` | 红色 / 金色 |
| 面朝内 | `Ω: 0.782` / `TARGET LOCK` | 白色 / 青色 |
| 左侧面 | `S: 47000 Du` / `ENTROPY` | 红色 |
| 右侧面 | `TIER-3: 34%` / `CAPACITY` | 黄色 |

4. 在告示牌上方和下方各放 1 个末地烛，提供照明使文字可读。
5. 告示牌两侧各放 1 面旗帜：
   - 红色底+黑色条纹旗帜（WARNING 位置）
   - 紫色底+白色十字旗帜（状态显示位置）

## 第十一部分：连接结构与辅助元素

### 11.1 恒星炉日冕弧→法阵连接

日冕弧的末端不是消散在空气中——最长的弧应延伸至法阵膜面的底面（左侧/橙红区域），并与之相接：

- 从炉 A 的最长日冕弧（A-3, 18格）末端延伸一条 5–8 格的橙色玻璃"尾巴"，连接到法阵膜面左侧的尖刺区域底面。
- 接合处使用荧石+橙色玻璃混合，制造明亮的能量注入点。
- 炉 B 镜像处理。

### 11.2 数据瀑布→螺旋光柱连接

浑天仪数据瀑布中最长的 3–4 条粗流，其末端应接近或触及螺旋光柱的上段（h ≈ 280–290 处）：

- 在瀑布末端与螺旋光柱之间保留 3–5 格间距（不直接接触），但用品红色玻璃板在间隙中架设几条"数据桥"——水平或微斜的单片玻璃板，暗示 Logos 场正在从计算核心向螺旋光柱注入秩序指令。

### 11.3 恒星炉→法阵→螺旋→约束环的能量流路径标记

为了让观察者能直观理解能量从恒星炉到约束环的完整路径，可以在路径上设置视觉引导标记：

1. **橙色引导线**：从恒星炉日冕弧末端起，沿法阵膜面底面向中心方向，用橙色玻璃板铺设一条 1 格宽的引导线，引向漏斗口。
2. **螺旋引导线**：在螺旋光柱中，螺旋臂 A（暖色系）从顶部到底部的连续性本身就是引导。
3. **白色引导线**：从螺旋光柱底部进入约束环上唇口时，在入口圈用白色混凝土描一圈高亮边框。

### 11.4 注入器→约束环的冲击效果增强

每把注入器的剑尖嵌入约束环冲击点后，在冲击点外围额外建造"伞状飞溅"：

- 从冲击点向外辐射 4–6 根"光屑轨迹"——每根由 3–5 格白色玻璃板沿螺旋衰减路径排列（向外+向下弯曲）。
- 轨迹末端放 1 个末地烛，表现光屑的最终消散点。
- 每个冲击点共 4–6 条轨迹，在冲击点外侧形成一个不完整的"伞盖"效果。

## 第十二部分：照明与特效设计

### 12.1 整体照明策略

"达摩克利斯"是一个内发光结构——其大部分光源来自结构内部，而非外部光照。Minecraft 的光照系统基于方块级光源（最大光照等级 15，每格衰减 1），因此需要在结构中密集嵌入发光方块。

**各区域光源密度建议**：

| 区域 | 主光源类型 | 密度 | 环境光照等级目标 |
|------|-----------|------|----------------|
| 火球表面 | 荧石（格元中心） | 每 5–7 格 1 个 | 15（火球表面应为最亮区域） |
| 日冕弧 | 荧石（弧内） | 每 3 格 1 个 | 12–15 |
| 浑天仪 | 末地烛（节点）+ 荧石（核心） | 每 4–6 格 1 个 | 10–13 |
| 数据瀑布 | 末地烛（瀑布内） | 每 5 格 1 个 | 8–11 |
| 法阵膜面（紫色区） | 紫水晶簇（六边形中心） | 每 8–10 格 1 个 | 5–8（应为柔和的紫色暗光） |
| 法阵膜面（橙红区） | 荧石 + 岩浆块 | 每 4–5 格 1 个 | 10–13（明亮但不均匀） |
| 螺旋光柱 | 海晶灯/荧石/末地烛（混合） | 每 3–4 格 1 个 | 12–15（中心明亮的发光柱） |
| 约束环 | 海晶灯（符文层） | 每 3–4 格 1 个 | 12–15（核心高亮区） |
| 白色光雾（环内腔） | 荧石（散布） | 约 30% 体积 | 15（纯白高亮） |
| 注入器 | 海晶灯（脉冲节点） | 每 4–5 格 1 个 | 8–12（渐变） |
| 巨剑金色段 | 荧石（内嵌） | 每 6–8 格 1 个 | 8–10（温暖金色辉光） |
| 巨剑灰白段 | 无（或极少末地烛） | — | 3–5（暗淡，仅靠环境光） |
| 发射井 | 岩浆块 + 红石灯 | 见2.1节 | 5–8（深红暗光） |
| 地面熵疤裂缝 | 岩浆块 + 红石灯 | 每条裂缝内连续 | 3–7（从裂缝中渗出的不安红光） |

### 12.2 信标的战略性使用

信标（Beacon）在本项目中有两个用途：

1. **浑天仪核心**（已在 9.2 节描述）：紫色信标光束从浑天仪中心向上投射，穿透结构顶部，成为远距离可见的标志性光柱。
2. **螺旋光柱核心辅助**（可选）：在螺旋光柱的中轴线上（约束环中心位置 h=158），放置第二个信标，光束向上穿过螺旋光柱内部。信标颜色设为白色或品红色（通过上方放置染色玻璃改变光束颜色）。这会在螺旋双臂之间的空隙中添加一道中央光柱，增强"能量通道"的视觉效果。

**信标底座隐藏**：信标需要 3×3 至 9×9 的金属方块底座。可将底座隐藏在约束环壁体内（10格壁厚完全足够容纳）或浑天仪核心体内。

### 12.3 粒子效果的方块近似

Minecraft 原版中无法直接创建自定义粒子效果，但以下方块和实体可以产生视觉上类似粒子的效果：

| 目标效果 | Minecraft 近似方案 |
|---------|-------------------|
| 白色光雾/光屑 | 白色玻璃（大面积）+ 灵魂灯笼（蓝白光点）+ 末地烛（竖直光棒） |
| 金色准晶闪烁 | 荧石（嵌入金块中）+ 金质压力板（表面装饰）+ 光源带来的高光反射 |
| 暗红色熵排放光斑 | 红色蜡烛（点燃）产生的小火焰粒子 |
| 紫色数据流 | 紫水晶芽/簇的自然光粒子 + 附魔台附近的浮动符文效果（如在数据瀑布附近放置附魔台和书架） |
| 岩浆泡沫 | 岩浆块的表面动态纹理 + 熔岩源方块的自然粒子 |
| 蒸汽/喷气 | 灵魂砂上的灵魂火焰（蓝色）/ 篝火（白烟）放置在冷却管道出口上方 |

**进阶（命令方块辅助）**：

如果允许使用命令方块，可以通过 `/particle` 命令在关键位置持续生成粒子：

```
# 约束环冲击点——白色光屑飞溅
/particle minecraft:end_rod <x> <y> <z> 2 2 2 0.05 50 force

# 恒星炉周围——火焰粒子
/particle minecraft:flame <x> <y> <z> 5 5 5 0.02 100 force

# 浑天仪——附魔粒子
/particle minecraft:enchant <x> <y> <z> 3 3 3 0.1 80 force

# 地面裂缝——暗红烟雾
/particle minecraft:crimson_spore <x> <y> <z> 1 0.5 1 0.01 30 force

# 螺旋光柱——灵魂火焰粒子（蓝色）
/particle minecraft:soul_fire_flame <x> <y> <z> 2 5 2 0.03 60 force
```

将这些命令放入循环命令方块（Repeating Command Block），可以产生持续的粒子效果。

## 第十三部分：建造顺序建议

大型结构的建造顺序对效率和精确度有重大影响。以下为推荐的建造序列：

### 阶段一：参考框架搭建

1. **中心标记**：在 (0, 0) 位置从 Y=46 到 Y=411 搭建一根临时标记柱（使用羊毛或混凝土，每 10 格换色标注高度）。
2. **关键高度标记**：在标记柱上用告示牌标注所有关键 h 值（h=18 地面、h=25 剑尖、h=150 约束环中心、h=168 光柱起点、h=310 漏斗口、h=325 法阵基准、h=332 浑天仪中心）。
3. **水平参考圈**：在 h=158 高度（约束环中心）放置一个临时的半径 26 格的圆形框架（用任意醒目方块），标记约束环外壁位置。在 h=325 高度放置另一个半径 110 格的临时圆标记法阵外缘。

### 阶段二：核心结构体

4. **约束环**（第四部分）：从中心向外建造。先建内壁白色玻璃层，再逐层向外添加符文层和外壁。
5. **巨剑**（第三部分）：从约束环内腔的 h=155 位置向下建造。逐层降低 h，按截面图放置方块，同时执行颜色渐变和纹路图案。
6. **发射井**（2.1 节）：从地面向下挖掘建造。

### 阶段三：辐射结构体

7. **8把注入器**（第五部分）：从约束环外壁的冲击点位置向外建造。先建正交方向（容易），再建对角线方向（需要处理锯齿）。
8. **螺旋光柱**（第六部分）：从约束环上唇口（h=168）向上建造。每层放置螺旋臂方块前，先在中心标记柱上对照当前应旋转的角度。

### 阶段四：上层结构体

9. **法阵膜面**（第七部分）：从中央漏斗口（h=310–318）向外扩展。先建中央过渡带（白色），再分别向左（橙红尖刺）和向右（紫色六边形）建造。
10. **双子恒星炉**（第八部分）：分别在 (−50, −35) 和 (+50, −35) 位置建造。先建火球球体（脚手架辅助），再添加机械爪和助推器。
11. **浑天仪**（第九部分）：在 (0, +45) 位置建造。先建外层环，再嵌套中层和内层。最后安装核心体和信标。

### 阶段五：连接与装饰

12. **紫色锁链**（8.6 节）：连接浑天仪和恒星炉暗斑。
13. **数据瀑布**（9.3 节）：从浑天仪底部向下悬挂。
14. **日冕弧**（8.3 节）：从火球表面向法阵中心方向延伸。
15. **外围装甲板**（第十部分）：在螺旋光柱和法阵外围散布放置。
16. **注入器冲击效果**（11.4 节）：在约束环冲击点外侧添加白色光屑轨迹。

### 阶段六：地面环境

17. **地面熵疤**（2.2 节）：在地面层进行大面积地表替换和裂缝开挖。
18. **悬浮碎石**（2.3 节）：在空中散布各尺寸岩石。
19. **冷却管道**（2.4 节）：从地面基座向地下三个方向建造。
20. **地面基座**（2.5 节）：围绕井口建造仪式平台。

### 阶段七：收尾

21. **照明补充**（第十二部分）：检查所有区域的亮度，在暗区补充光源方块。
22. **粒子效果**（12.3 节）：放置命令方块（如使用）。
23. **告示牌与旗帜**（10.7 节）：安装全息文字模拟。
24. **拆除临时结构**：移除中心标记柱、所有临时脚手架和参考框架。

## 第十四部分：高级建造技巧与常见问题

### 14.1 大型球体的建造

恒星炉火球（直径 26 格）和浑天仪环体都涉及球面建造。推荐方法：

1. **Plotz 球体生成器**（在线工具）：输入直径，获取每层的圆形切面图案。逐层搭建。
2. **WorldEdit //hsphere 命令**：`//hsphere orange_concrete 13` 生成空心球壳，然后手动进行表面纹理替换。
3. **手动逐层法**：在中心高度画最大圆（直径=球体直径），然后向上和向下逐层缩小圆的直径（按球体截面公式）。

### 14.2 环面体（Torus）的建造

约束环是一个大型环面体。建造方法：

1. 确定环面的两个参数：大半径 R=21 格（环心到截面中心的距离）、小半径 r=5 格（截面半径）。注意：外半径 = R+r = 26，内半径 = R−r = 16。
2. 在每个角度 θ（0° 到 360°，步进约 5–10°）上，在 (R·cosθ, R·sinθ) 位置画一个半径 r 的圆形截面。
3. 由于每一层（每个 h 值）是一个水平切面，实际建造时是逐层放置同心圆环——每层的外半径和内半径根据该层距环心高度的距离而变化。

**简化方法**：用 WorldEdit 的 `//torus` 命令（如有插件支持），或使用在线环面体生成器获取每层的圆环图案。

### 14.3 螺旋线的精确建造

双螺旋结构是本项目中建造难度最高的部分之一。以下是逐层参数化方法：

设螺旋在某高度 h 的参数为：
- 当前半径 `r(h)`（见 6.2 节的线性插值）
- 当前旋转角度 `θ(h) = θ₀ + Σ(Δθ)`，其中 Δθ 每层约 14°–24°

在每一层 h 上：
1. 计算 `r(h)` 和 `θ(h)`
2. 螺旋臂 A 的中心坐标：`(r·cos(θ), r·sin(θ))`
3. 螺旋臂 B 的中心坐标：`(r·cos(θ+180°), r·sin(θ+180°))`
4. 在各中心坐标处放置 2–3 格宽的弧段方块

**WorldEdit 辅助**：可以用 `//curve` 命令定义螺旋线上每隔 10 层的关键点，用三次样条插值自动填充中间方块。

### 14.4 对角线结构的锯齿优化

对角线方向的注入器和装甲板会出现明显的锯齿阶梯。优化技巧：

1. **加宽截面**：将 3 格宽的截面改为 4 格宽，使阶梯转折处有更多方块可用于平滑。
2. **交替偏移**：在每个阶梯步进中，交替将截面在宽度方向偏移 0.5 格（用半砖实现）。
3. **玻璃板过渡**：在阶梯内角处放置玻璃板（45° 朝向），填充视觉空隙。
4. **颜色掩饰**：在阶梯步进处使用比主体颜色略暗/略亮的方块，利用色差来弱化阶梯的视觉影响。

## 第十五部分：可选增强功能

### 15.1 红石驱动的"以太呼吸"系统

**目标**：巨剑表面的金色纹路周期性明暗变化。

**实现方案**：

1. 在巨剑金色段（h 80–130）的纹路交汇节点处，将原本的荧石替换为红石灯。约需 6–10 个节点。
2. 在血槽内部（空腔空间）铺设红石线路，连接所有节点上的红石灯。
3. 在血槽内某个位置放置一个漏斗时钟（Hopper Clock），周期约 48 红石刻（≈ 4.8 秒，接近 2×φ² ≈ 5.24 秒的以太呼吸周期）。
4. 时钟信号通过红石线路同步驱动所有红石灯，使纹路节点同时亮灭交替。

**漏斗时钟构造**（放置于剑身内部）：

```
两个漏斗面对面相连，其中一个漏斗内放置若干物品。
物品在两个漏斗之间来回传递，每次传递花费约 0.4 秒/物品。
放入 12 个物品 → 单程传递约 4.8 秒 → 一个完整周期约 9.6 秒。
从中间取比较器信号即可获得约 4.8 秒的半周期脉冲。
```

### 15.2 注入器脉冲传播动画

**目标**：每把注入器上的 8 个脉冲节点依次闪亮，模拟寒极脉冲从剑柄向剑尖传播。

**实现方案**：

1. 将注入器的 8 个脉冲节点处的海晶灯替换为红石灯。
2. 在注入器内部（截面 3–5 格，可利用中心空腔）铺设红石线路，使用中继器 (Repeater) 链实现顺序延迟。
3. 每两个相邻节点之间的中继器设为 2 刻延迟 → 8 个节点总传播时间 = 14 刻 ≈ 1.4 秒。
4. 在最后一个节点处连接线路回到第一个节点（循环），但中间插入一个 20 刻的"冷却"延迟 → 完整周期 ≈ 3.4 秒。

**挑战**：注入器截面较细（3–5 格），红石线路的空间非常有限。对角线方向的注入器尤其困难。建议仅在正交方向的 4 把注入器上实现此效果。

### 15.3 浑天仪"共振闪光"

**目标**：浑天仪上的 12 个交叉节点同步闪烁。

**实现方案**：

1. 在 12 个节点处放置红石灯（替代末地烛簇中的 1–2 个末地烛）。
2. 在浑天仪核心体内放置一个红石时钟（周期约 10 刻 ≈ 1 秒）。
3. 从核心向外沿环体内部走红石线路至 12 个节点。
4. 由于环体截面为 2×2 或 3×3 格，红石线路可隐藏在截面内部。

### 15.4 音效提示（使用音符盒）

在约束环附近放置几个音符盒（Note Block），调音至高音域（铁块底座 = 铁琴音色），模拟"金属颤鸣"的氛围音效。使用红石时钟驱动，以不均匀的间隔触发，营造不安的工业魔法氛围。
