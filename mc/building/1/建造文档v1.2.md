# "达摩克利斯"天基铸剑阵列 —— Minecraft 精细化建造设计文档（修订版）

## 第〇章：总则

### 0.1 尺度定义

| 范围 | 区块数 | 格数 | 用途 |
|------|--------|------|------|
| 主体结构区（空置域） | **17×17 区块** | **272×272 格** | 所有功能性结构体（巨剑、约束环、恒星炉、浑天仪、法阵、注入器、螺旋光柱）的包络范围 |
| 外围环境区 | 33×33 区块 | **528×528 格** | 地面熵疤、悬浮碎石场、冷却管道、远场环境效应、外围装甲板最外缘 |

**核心设计原则**：所有主要结构组件必须落在 272×272 格的正方形空置域内。装甲板的散布弧段和地面环境效应可以延伸到 528×528 格的外围区域。

### 0.2 坐标与朝向约定

| 约定 | 定义 |
|------|------|
| 中轴线 | X = 0, Z = 0，巨剑轴心线，垂直贯穿整个结构 |
| 朝向 | 面向 **−Z 方向**为"正面"（站在 +Z 侧，向 −Z 方向观看结构正面） |
| Y 轴 | 向上为正。**地面基准 Y = 64**（标准主世界地面） |
| 相对高度 h | h = Y − 44。结构最低点（发射井底部）在 h = 0（Y = 44） |
| 空置域半轴 | X 方向 ±136 格，Z 方向 ±136 格（对称正方形） |

- 五角星不需要精确完美——以太准晶的衍射花纹本身就是"近似五重对称"而非精确对称。适当的不规则性反而更符合设定。

在巨剑宽度达到11格以上后（h ≥ 55），刃缘处理变得尤为重要。

截面刃缘剖面（沿X方向的最外2格）：
```
正面(−Z)视角下，最外端2格的侧剖面：

     ┌─────┐ 
     │     │  ← 实心方块（层3-5中心区）
    ╱│     │╲
```

**Minecraft实现**：

- 最外列（刃尖）：使用材质对应的台阶方块（方解石→黄色陶瓦→金块，随高度渐变）。上半台阶和下半台阶交替放置，使11层厚的截面在最外端削薄为约6层的视觉高度。
- 次外列：使用全方块，但在上表面（+Y面）和下表面（-Y面）各放一个台阶方块（底部形态），进一步增加从侧面观看时的锐角感。

沿剑身两侧刃缘最外1格的外侧，每隔3–4格放置1个白色玻璃板（垂直放置于空气中，紧贴刃缘）。在金色段（h 80–140），玻璃板发出的半透明白色与金色方块的反射光叠加，形成柔和的"界面辐射"光晕。在灰白段（h 30–55），白色玻璃板与剑身颜色接近而不显眼。

**光晕密度随高度递增**：

- h 30–80：每6格1个玻璃板（微弱光晕）
- h 80–130：每3格1个（中等光晕）
- h 130–165：每2格1个（强光晕，叠加白雾效果）

### 3.7 "以太呼吸"红石系统

| 参数 | 数值 |
|------|------|
| 节点数量 | 每面8–10个（两面合计16–20个） |
| 节点位置 | h 85–135范围内，均匀分布于五角纹路图案的中心或交汇点 |
| 节点方块 | 红石灯（替代该位置的荧石） |
| 脉冲周期 | 48红石刻 ≈ 4.8秒（近似 2×φ² ≈ 5.24秒） |
| 时钟位置 | 血槽内部 h ≈ 110 处 |

**详细红石线路布局**：

血槽A（+Z面）纵剖面示意：
```

             ↑              ↑              ↑
h=120:  ════╤══════════════╤══════════════╤══  ← 红石主线（中继器延伸）
             │              │              │
```

**漏斗时钟构造（放置于血槽内部，约2×4×2格空间）**：
```
┌────────────┐
│  H ← → H  │    H = 漏斗（面对面相连）
│  ↑ items ↓ │    其中一个漏斗内放12个物品
│  C         │    C = 比较器（从漏斗取信号）
│  ↓         │
│  红石粉 →  │    信号输出至红石主线
└────────────┘

12个物品在2个漏斗间往返，单程约4.8秒。
```

**信号从血槽到外表面的穿透**：

在每个红石灯节点位置，血槽内壁有一个1格缺口，红石线路通过此缺口将信号传递到嵌入剑身外表面的红石灯。红石灯位于五角纹路图案的中心位置，替代原本的荧石方块。

当红石灯ON时，纹路中心发出强烈红石灯光（光照等级15）；当OFF时，该位置暗淡。配合周围持续发光的荧石方块，形成节点"明亮→回暗"的呼吸脉动效果。

### 3.8 剑身完成后视觉效果

从发射井口向上仰望：

> 一把巨大的实体剑从高处直刺而下，剑尖距离井口仅10格。灰白色的剑尖如磨制的石柱，致密、冷硬、毫无光泽——这是秩序指数最高、物质密度最大的区域。

**逐层截面半径表（中部15层示意）**：

### 5.6 对角线方向注入器的建造

注入器 #1, #3, #5, #7 沿45°对角线方向延伸，在Minecraft方块网格中产生阶梯锯齿。

**阶梯步进规则（以#1，西北45°方向为例）**：
```
每前进1格 −X 同时前进1格 −Z。
即对角线上的每一步 = 坐标变化 (ΔX=−1, ΔZ=−1)。

实际建造时，注入器的"长轴"沿对角线前进，
但每一步的截面需要旋转45°以适应方块网格。
```

**优化阶梯感的具体方法**：

1. **截面加宽**：对角线注入器的截面从7格宽改为8格宽（偶数宽度在对角线上更好对齐）。

2. **交替偏移法**：
    ```
    步进序列（俯视，沿对角线方向前进，以#1西北45°为例）：

     步骤1: 放置截面于 (X, Z)
     步骤2: 前进至 (X−1, Z−1)，截面在X方向偏移+0.5格（用半砖实现）
     步骤3: 前进至 (X−2, Z−2)，截面回到标准位置
     步骤4: 前进至 (X−3, Z−3)，截面在Z方向偏移+0.5格
     步骤5: 前进至 (X−4, Z−4)，截面回到标准位置
     循环以上模式……

    实际操作——每"步"的截面放置：

       步骤N（标准位）:          步骤N+1（X偏移位）:
       
       · ℓ B B B B ℓ ·          · · ℓ B B B B ℓ ·
       ℓ B B B B B B ℓ          · ℓ B B B B B B ℓ
       · ℓ B B B B ℓ ·          · · ℓ B B B B ℓ ·

       对角线上每一步之间存在1格的阶梯落差，
       通过交替将截面在X或Z方向偏移半格（实际使用半砖/台阶），
       将原本的90°直角阶梯软化为约135°的钝角过渡。
    ```

3. **进一步减轻锯齿的技巧**：
    - **对角截面旋转**：对角线注入器的截面不使用与坐标轴对齐的矩形，而是将整个截面旋转45°——即菱形朝角放置。这使得截面的最长对角线沿前进方向排列，自然减少突兀感。
        ```
        正交注入器截面:          对角线注入器截面:
                         
           · ℓ B B B ℓ ·              · · · ℓ · · ·
           ℓ B B M B B ℓ              · · ℓ B ℓ · ·
           · ℓ B B B ℓ ·              · ℓ B B B ℓ ·
                                       ℓ B B M B B ℓ
           宽7×厚3                    · ℓ B B B ℓ ·
                                       · · ℓ B ℓ · ·
                                       · · · ℓ · · ·
                                       
                                       旋转45°后的菱形截面
                                       对角线宽7格，正交宽约5格
        ```
    - **阶梯转折处的玻璃板填充**：在每一步阶梯的内角（即两个方块层之间90°转角处），放置淡蓝色玻璃板（45°朝向），填充视觉空隙。玻璃板的半透明属性使其不会完全遮挡背后的天空，但能有效柔化硬边轮廓。
    - **颜色掩饰**：在阶梯步进处使用比主体材质略暗的方块变体——例如在冰蓝段的阶梯转折处用蓝色混凝土替代浅蓝色混凝土（暗1级），利用色差使视线自然跳过阶梯边界。
    - **WorldEdit/Axiom 辅助**：对于有辅助工具的建造者，推荐使用 //line 命令从剑柄中心坐标到约束环冲击点中心坐标画一条直线作为中轴参考，然后手动在中轴线周围搭建截面。

**对角线注入器的护手处理**：

对角线方向的十字护手同样需要旋转45°。原本沿X和Z轴展开的护手臂，旋转后沿 (X+Z) 和 (X−Z) 方向展开：
```
对角线注入器的护手俯视（以#1西北方向为例）：

                · D ·
              · D D D ·
            D D D D D D D        ← 护手臂沿(−X+Z)和(+X−Z)方向展开
          · D D D █ D D D ·          总宽仍22格（沿旋转后的对角线测量）
            D D D D D D D            实际占据约16×16格的方块范围
              · D D D ·
                · D ·

由于对角线方向的护手臂沿方块对角线伸展，
每一格护手臂实际占据1个方块的对角线位置，
使用台阶/半砖在对角方向叠出3格宽的臂身。
```

### 6.3 螺旋臂的逐层建造方法

每一层(每个h值)的操作流程：
```
步骤1: 查表获取当前层的半径 r(h) 和旋转角度 θ(h)。

步骤2: 在该层的水平面上，以 (0, 0) 为圆心，画出半径 r(h) 的圆
       （可用Minecraft圆形图表，或WorldEdit //hcyl）。

步骤3: 在圆周上找到角度 θ(h) 的位置——这是螺旋臂A的中心点。
       坐标计算：
           X_A = r × cos(θ)    （注意Minecraft中X轴与角度的对应）
           Z_A = r × sin(θ)    （可能需要根据朝向约定调整正负号）
       
       取整至最近的方块坐标。

步骤4: 以臂A中心点为基准，沿圆弧方向（切线方向）铺设 臂宽 格数的方块弧段。
       弧段厚度2格（当前层h放1层方块，下方h−1已有的前一层方块构成2格厚度）。

步骤5: 在 θ + 180° 位置重复步骤3–4，放置螺旋臂B。

步骤6: 在两条螺旋臂之间的空间中，按色彩分区表放置填充玻璃方块。
```

**臂弧段的方块排列（单层示例，r=18，臂宽4格）**：
```
在半径18格的圆周上，臂A中心角度 θ = 345°:

圆周上 θ = 345° 的坐标: X ≈ +17, Z ≈ −5

臂弧段沿圆弧方向（逆时针为正）跨越约 4/(2π×18) × 360° ≈ 12.7° 的弧度。
即从 θ−6° 到 θ+6° 的弧段。

实际放置：在 θ=339° 到 θ=351° 的弧线上，
          逐格放置对应色的方块。由于弧线在方块网格上的投影，
          实际会放置 4–6 格方块形成一段弧。
```

**WorldEdit 辅助建造命令序列**：

### 6.7 螺旋光柱完成后的视觉效果

从地面（发射井口）向上仰望：

> 巨剑消失在约束环的白色光雾中。光雾之上，约束环的上唇口收束为一个直径60格的收口，从这个收口中，一根巨大的螺旋光柱拔地——不，拔天而起。

> 光柱底部是白蓝紫三色交织的双螺旋——两条螺旋臂如同DNA的双螺旋结构般紧密缠绕，一条偏蓝白、一条偏品红紫色，它们之间的空间被半透明的白色玻璃填充，形成一个约18格粗的发光柱体。白色的核心柱如同光柱的脊椎贯穿中心。

> 再向上，品红开始分裂——一条臂恢复为橙色（原火的痕迹），另一条臂恢复为紫色（理式的回归）。两种颜色交替出现，在中心区域混合为斑驳的暖紫色光斑。螺旋继续展宽至50格直径。

> 最顶部，螺旋光柱骤然张开为一个90格直径的喇叭形漏斗口。漏斗壁上的橙紫螺旋条纹加宽加粗，内壁则是均匀发白光的海晶灯面。漏斗口的顶缘由一圈海晶灯镶边——这道明亮的白色环线标记着螺旋光柱与法阵膜面的分界。

### 7.5 中央过渡带（沿 X ≈ 0 附近蜿蜒，Z方向延伸）

这条过渡带是整个法阵膜面视觉上最引人注目的分界线——秩序与混沌在此交锋。

**几何参数**：
```
中心线：大致沿 X = 0，但蜿蜒偏移——
    Z 每隔 20–30 格，X 偏移 ±3–8 格。
    蜿蜒的方向不规则——有时向+X（紫色区域侵入），
    有时向−X（橙色区域反推）。
    
过渡带宽度：15–25格（不固定）。
    在紫色区域强势的段落，宽度窄至15格；
    在橙色区域反推的段落，宽度扩至25格。

高度：h = 340–350。
    低于右侧膜面平台（h≈348）约2–4格，
    远低于左侧尖刺峰顶（h=355–365）。
    这使得过渡带看起来像一条低洼的河谷，
    两侧分别是紫色的有序高原和橙色的混沌山脊。
```

**过渡带建造**：
```
主体材质：白色混凝土(55%) + 淡灰色混凝土(25%) + 海晶灯(20%)。

厚度：4格（h=340–343 或对应段的底部4层）。

上表面处理（h=343 或对应顶层）：
    白色混凝土平铺，极度光滑。
    每隔10–12格沿过渡带中心线放置1根末地烛
    （竖直指向天空），形成一排白色光标哨兵。
    共约 20–25 根末地烛排成蜿蜒的中心线。

"侵入物"——秩序侵入混沌：
    在过渡带靠近左侧（−X方向）的边缘，

        2×2格品红色混凝土方块 + 中心1个紫水晶芽。

    这些前哨暗示理式秩序场正在试图侵蚀混沌区域的边缘。

"侵入物"——混沌侵入秩序：
    在过渡带靠近右侧（+X方向）的边缘，
    每隔15–20格嵌入一组"橙色锲入"：
        3×1格橙色玻璃方块 + 1个岩浆块（底部嵌入白色地面中）。

底面（朝下的面，h=340层）：
    白色玻璃(70%) + 海晶灯(30%)。

```

**过渡带与两侧的接缝处理**：

**右侧接缝（过渡带→紫色六边形区域）**：
```
过渡带的 +X 边缘（X ≈ +5 至 +15）:
```

| 亮格元比例 | 约25–30%（温度更高的对流上升区） |
| 标准格元比例 | 约55–60% |
| 暗格元比例 | 约10–15%（温度较低的下沉区,靠近暗斑） |

**格元建造方法（分三步）**：

**第一步：铺设底色球壳**
```
用橙色混凝土建造完整的60格直径空心球壳（壳厚2格）。
内壳层：橙色混凝土（内层，提供实心感）。
外壳层：橙色混凝土（外表面，将被格元纹理覆盖）。
球体内部空心（但会在后续步骤中填充部分发光方块）。
```

**第二步：绘制格元边界网络**
```
在球面外壳层(最外层)上，用橙色陶瓦+红色陶瓦绘制不规则多边形网格：

1. 从球体的"赤道"（中心高度层）开始，沿水平方向每隔8–12格
   画一条2格宽的竖直红色陶瓦条带（从赤道向两极延伸约15–20格）。
   共约12–16条竖直条带，不均匀分布。

2. 沿垂直方向每隔8–12格画一条2格宽的水平橙色陶瓦环带。
   共约6–8条环带。

3. 竖直条带与水平环带的交叉点即为格元的"顶点"。
   在顶点之间沿球面画出不规则连线（不要走直线，
   应有1–2格的随机偏移），形成3–6边形的不规则格元。

4. 边界材质分配：约60%使用橙色陶瓦，40%使用红色陶瓦。
   较暗的红色陶瓦集中在暗斑附近的格元边界。
```

**第三步：格元内部材质分配**
```
对每个格元（由边界围成的多边形区域）进行内部材质替换：

亮格元（25–30%，集中在面向法阵中心的半球前方）：
    橙色混凝土底色 → 替换中心60%区域为黄色混凝土
    格元中心放置1个荧石方块（发光核心）
    格元中心外围放置1–2个黄色混凝土
    格元边缘保持橙色混凝土（与边界过渡）

标准格元（55–60%）：
    保持橙色混凝土底色
    格元中心放置1个荧石方块（标准发光）
    无额外替换

暗格元（10–15%，集中在暗斑周围和极区）：
    橙色混凝土底色→部分替换为棕色混凝土（暗区标志）
    格元中心放置1个岩浆块（代替荧石，更暗的发光+动态纹理）
    格元边缘用灰色陶瓦替代部分橙色混凝土
```

**球体内部发光支持**：

由于球壳厚度仅2格，荧石（光照15）在球面上可能无法充分照亮周围表面。需要在球壳内层额外嵌入发光方块：
```
在球壳内层（第2格，从外往内数），每隔4–5格嵌入1个荧石方块。
这些内层荧石不从外部可见，但提供充足的间接光照，
使球面上所有格元即使在夜晚也清晰可见。
内层荧石总量约 200–250个。
```

### 9.1 总体参数

浑天仪位于法阵膜面的 +Z（背面/正后方）方向，与两座恒星炉呈倒三角形布局（两炉在前方−Z侧，浑天仪在后方+Z侧）。

| 参数 | 数值 |
|------|------|
| 中心坐标 (X, Z) | **(0, +90)** |
| 中心高度 | **h = 354**（Y = 398） |
| 外层球体直径 | **80格**（半径40格） |
| 中层球体直径 | **50格**（半径25格） |
| 内层球体直径 | **30格**（半径15格） |
| 直径比 | 80 : 50 : 30 ≈ 1 : 0.625 : 0.375 ≈ 1 : φ⁻¹ : φ⁻¹·φ⁻¹ |
| 最高点 | h = 354 + 40 = **394**（外层球顶，但被截断——见下文） |

**空间校验**：

| 校验项 | 数值 | 安全? |
|--------|------|-------|
| Z方向极端值 | +90 + 40 = +130 | < 136 ✓（裕度6格） |
| Z方向近端 | +90 − 40 = +50 | 不与巨剑轴心冲突 ✓ |
| X方向极端值 | 0 ± 40 = ±40 | < 136 ✓ |
| 顶部 h | 394 | > 378（结构名义最高点），可截断顶部16格或将中心下移至h=346 |

**高度调整方案**：将浑天仪中心设为 **h = 346**（Y = 390），则外层球顶 h = 386，底部 h = 306。球顶微超378的名义限制8格，但仍远在Minecraft建造高度上限（Y = 319+扩展 或创造模式更高限制）之内。或保持 h = 354 不变，外层球体顶部15格区域不建造完整球壳，而是以开裂/数据流溢出的形式处理——球体顶部是"开放"的，紫色数据瀑布从开口向下流淌。

**推荐方案**：保持中心 h = 354，外层球体顶部做**截顶处理**——不封闭球顶，而是在 h = 380 处截断（距球顶保留约14格的开口），开口处放置紫色玻璃板和紫水晶簇作为"数据溢出口"。这同时解决了高度问题和提供了视觉特色。

### 9.2 三层浑天仪的整体结构

浑天仪由三层同心但不同轴倾斜的球壳环组成，模拟古代浑天仪（Armillary Sphere）的多轴旋转结构。

**三层定义**：

| 层 | 名称 | 直径(格) | 壳厚(格) | 环宽(格) | 倾斜轴 | 倾角(°) | 材质色系 |
|----|------|---------|---------|---------|--------|--------|---------|
| 外层 | 黄道环 (Ecliptic Ring) | 80 | 2 | 6 | Y轴（竖直） | 0（不倾斜） | 紫珀块+紫珀柱 |
| 中层 | 赤道环 (Equatorial Ring) | 50 | 2 | 5 | X轴 | 23.5° | 紫水晶块+紫珀块 |
| 内层 | 子午环 (Meridian Ring) | 30 | 2 | 4 | Z轴 | 45° | 紫水晶块+紫水晶簇 |

**注意**：这三层不是完整的实心球壳——它们是**环体**，即沿各自的赤道方向有一条宽带是实心的（环宽），而其余部分是空气。可以理解为三个正交的巨大圆环套在一起。

### 9.4 中层：赤道环 (Equatorial Ring) — 直径50格

**倾斜圆环的建造方法**：

Minecraft中建造倾斜圆环是一个挑战。推荐以下方法：
```
方法A：参数方程逐格定位

赤道环的中心点：C = (0, +90, h=354) → 世界坐标 (0, 398, 90)

对于倾斜角α=23.5°绕X轴旋转的半径R=25的圆：
    X(t) = 0 + R × cos(t)
    Y(t) = 398 + R × sin(t) × cos(α)
    Z(t) = 90 + R × sin(t) × sin(α)
    
    其中 t 从 0° 到 360°（步进约1°，即约360个采样点）。

    cos(23.5°) ≈ 0.917
    sin(23.5°) ≈ 0.399

代入R=25：
    X(t) = 25 × cos(t)
    Y(t) = 398 + 22.9 × sin(t)         ← Y方向被压缩
    Z(t) = 90 + 10.0 × sin(t)          ← Z方向微偏

每个采样点四舍五入到最近整数坐标，即为环体中心线的方块位置。

在每个中心线方块周围，按5×5截面放置方块
（需要根据局部环切线方向旋转截面朝向）。

方法B：WorldEdit/Axiom辅助（推荐）

1. 先在水平面建造一个外径50、内径40、高5格的标准水平圆环。
2. 使用 //rotate 命令绕X轴旋转23.5°。
   命令示例：
   //rotate 23.5 0 0
   （WorldEdit的旋转可能不支持非90°增量——
     此时使用Axiom工具或VoxelSniper的旋转功能）。
3. 旋转后移动到正确位置。

方法C：手动分层近似

将23.5°倾斜环分解为逐层的水平截面：
    倾斜环在每个Y层的截面是一个椭圆弧段。
    
    Y = 398 + Δy 时（Δy从−23到+23）：
    对应 sin(t) = Δy / 22.9 → t = arcsin(Δy/22.9)
    该Y层的截面 X 范围 = ±25×cos(t) = ±25×√(1−(Δy/22.9)²)
    Z 偏移 = 90 + 10.0 × Δy/22.9
    
    逐层绘制该椭圆弧段的5格宽壳体。
```

**材质分配**：
```
外壁（2格）：紫水晶块（Amethyst Block）
```

**弧线建造方法**：
步骤1：确定弧线参数
在俯视图中，确定该板片的圆心角 θ₁ 至 θ₂（例如 45° 至 65°）。
设半径 R = 195（内环中央值）。

步骤2：逐列放置

X(θ) = R × cos(θ)
Z(θ) = R × sin(θ)
每个位置放置一列高15–25格的弧壁竖切面。

步骤3：厚度方向
弧壁的厚度沿径向（指向中心的方向）展开。
每个弧线位置的"内/外"方向 = 从该点指向(0,0)的单位向量。

简化方法：使用WorldEdit的 //cyl 命令生成两个同心圆柱

**边缘处理**：
上边缘（板顶）：
最上2层使用台阶替代完整方块，形成楔形收薄。

### 12.4 动态光影与Shader推荐

本结构在原版光照下效果可观，但配合光影(Shader)可获得飞跃性的视觉提升：

✓ 反射（Reflections）：玻璃方块反映周围光源的倒影
✓ 动态阴影：装甲板在主体结构上投射阴影

推荐Shader包（截至2026年的主流选择）：
BSL Shaders — 综合性能与画质的平衡选择
Complementary Shaders — 色彩还原优秀，泛光效果突出
SEUS PTGI — 路径追踪级别画质，但性能要求极高
Iris + Sodium — Fabric端优化方案

注意：Shader环境下，发光方块的放置密度可适当降低（约减少30%），
因为泛光效果会自动扩大光源的视觉覆盖范围。

### 14.1 大型球体的建造

恒星炉火球（直径60格）和浑天仪核心球（直径12格）均涉及球面建造。
方法A：Plotz球体生成器（在线工具）
访问 plotz.co.uk 或类似网站，输入半径30，
获取每层的圆形切面图案（像素图）。
从赤道层（最大圆）开始，向上和向下逐层搭建。
每层打印/记录对应的圆形图案。

方法B：WorldEdit //hsphere命令
//pos1 （站在球心位置）
//hsphere orange_concrete 30
生成半径30格的空心橙色混凝土球壳。
然后手动进行表面纹理替换（格元、暗斑等）。

方法C：Axiom工具
Axiom mod提供可视化的球体笔刷，
可以直接在世界中"画"出球体。
支持实时预览和精确的半径控制。

方法D：手动逐层法（无工具可用时）
对于半径R的球，在距球心Δy层处的截面半径为：
r(Δy) = √(R² − Δy²)
例如R=30：
Δy=0 → r=30.0（赤道，最大圆）
Δy=5 → r=29.6
Δy=10 → r=28.3
Δy=15 → r=25.9
Δy=20 → r=22.4
Δy=25 → r=16.6
Δy=28 → r=10.8
Δy=29 → r=7.7
Δy=30 → r=0（极点）

### 14.2 环面体（Torus）的建造

约束环是一个大型环面体（外径120格、内径74格、高40格）。
参数定义：
大半径 R = (120+74)/4 = 48.5格（环心到截面中心的距离）
注：这里 R 和截面形状实际上更像方形截面环体
（因为壁厚23格×高40格的截面不是圆形而是矩形）。
对于方形截面环体，建造更直观：
建造方法：

在每个角度θ（0°到360°，步进1°）上：
计算该角度的中心位置：
X = 48.5 × cos(θ) （或直接使用内径37和外径60的差集）
Z = 48.5 × sin(θ)
在该位置放置一个23格宽×40格高的竖直矩形截面。

简化方法（推荐）：
使用WorldEdit建造两个同心圆柱（外圆柱半径60、内圆柱半径37），
取差集，高度40层：
//cyl deepslate_bricks 60 40 （外圆柱）
//cyl air 37 40 （挖掉内圆柱）
然后进行内部分层材质替换。

分层替换（在WorldEdit圆柱基础上）：
壁厚23格 = 外径60 − 内径37。
从内到外的圆柱壳层：
半径37–39（3格）：白色玻璃 + 荧石（内壁+光雾层）
半径39–42（3格）：下界砖 + 海晶灯（内过渡层+符文带）
半径42–51（9格）：铁块/深板岩混合（主结构层）
半径51–55（4格）：深板岩砖 + 海晶灯（外符文层）
半径55–60（5格）：深板岩砖/瓦（外壁层）

### 14.3 螺旋线的精确建造

双螺旋结构是本项目中建造难度最高的部分之一。
参数化方法（推荐，配合电子表格）：

在电子表格（Excel/Google Sheets）中创建一个表格：
列A：层号（h = 200 到 h = 320，共121层）
列B：当前半径 r(h)（从9格线性增长到25格）
r(h) = 9 + (25−9) × (h−200) / (320−200)
= 9 + 16 × (h−200) / 120
列C：当前旋转角度 θ(h)
螺旋总旋转约8–10圈（2880°–3600°），
每层旋转 Δθ = 3600° / 120 = 30°
θ(h) = 0° + 30° × (h − 200)
列D：螺旋臂A中心X = r × cos(θ)
列E：螺旋臂A中心Z = r × sin(θ)
列F：螺旋臂B中心X = r × cos(θ+180°)
列G：螺旋臂B中心Z = r × sin(θ+180°)

在游戏中，逐层对照表格中的坐标放置方块。
每层在螺旋臂中心坐标放置3–4格宽的弧段。

弧段方向：沿切线方向展开。
切线方向约为(−sin(θ), cos(θ))，
弧段的3–4格宽度沿切线方向展开。

WorldEdit辅助方法：
使用//curve命令，定义螺旋线上每隔5–10层的关键控制点：
//curve white_concrete <x1>,<y1>,<z1> <x2>,<y2>,<z2> ... <xN>,<yN>,<zN> 3
最后的"3"表示线条粗细为3格。
需要准备大量控制点坐标——建议用Python脚本预先计算并生成
WorldEdit命令文本，直接粘贴到聊天栏执行。

**Python辅助脚本示例**：

```python
import math

# 生成螺旋线控制点
points = []
for h in range(200, 321, 5):  # 每5层一个控制点
    r = 9 + 16 * (h - 200) / 120
    theta = math.radians(30 * (h - 200))
    x = round(r * math.cos(theta))
    z = round(r * math.sin(theta))
    y = h + 44  # h转换为Y坐标（Y = h + 44）
    points.append(f"{x},{y},{z}")

# 输出WorldEdit //curve命令（可能需要分段，因为//curve有节点数限制）
# 每20个点一条命令
for i in range(0, len(points), 20):
    chunk = " ".join(points[i:i+20])
    print(f"//curve white_concrete {chunk} 3")
```

### 14.4 对角线结构的锯齿优化
对角线方向的注入器（#1, #3, #5, #7）和对角装甲板会出现视觉锯齿。
优化技巧：

1. 加宽截面
   将11×5格截面改为12×6格，使锯齿转折处有更多方块可平滑。
   额外的1格宽度提供了在阶梯步进处放置过渡方块的空间。

2. 交替偏移（半砖技巧）
   在45°对角线方向前进时，每2格前进(+1X, +1Z)：
   奇数步：主截面完整放置
   偶数步：主截面在一个方向偏移1格，
           偏移产生的空隙用台阶/半砖填补。
   效果：锯齿从"2格大阶梯"变为"1格小阶梯+半砖过渡"。

3. 玻璃板过渡
   在阶梯内角处（锯齿的凹角）放置玻璃板（45°朝向）。
   玻璃板的1/8格厚度刚好填充凹角的视觉空隙。
   使用对应材质颜色的染色玻璃板（如白色玻璃板、蓝色玻璃板）。

4. 颜色掩饰
   在阶梯步进格使用比主体颜色略暗或略亮的方块变体。
   例如主体用白色混凝土，步进格用淡灰色混凝土——
   色差在远距离观察时模糊了阶梯的视觉轮廓。

5. 刻意利用锯齿
   对于注入器这种"剑形"结构，锯齿可以被重新解读为剑身锻打痕迹
   或表面凹凸纹理。不必追求完美平滑——Minecraft的方块美学
   本身就包含对非对齐的接受。

### 14.7 常见问题与解决方案

**Q1: 恒星炉火球看起来太"方"了，不像球体。**
A1: 60格直径的球体在Minecraft中已经相当圆。如果仍觉得不够：
    - 在球面上的"台阶"转折处放置台阶方块减轻1格阶梯
    - 使用格元边界纹理来分散视觉注意力（不让观者聚焦于球面曲率）
    - 在Shader环境下，体积光和泛光会柔化方块边缘

**Q2: 螺旋光柱的双螺旋看起来不连续/断断续续。**
A2: 螺旋臂的宽度需要足够：
    - 在底部（半径9格），每层螺旋弧段至少3格宽
    - 在顶部（半径25格），每层弧段至少5格宽
    - 如果旋转步进太大（每层>30°），会造成相邻层的弧段不重叠
    - 解决方案：减小步进角度（增加螺旋圈数），或增加弧段宽度

**Q3: 法阵膜面太大了，建造时迷失方向。**
A3: 将法阵膜面分为8个45°扇区，每次只建造一个扇区。
    在每个扇区边界放置临时彩色标记线。
    使用WorldEdit的//sel poly（多边形选区）限定工作范围。
    频繁飞到高空俯视检查对称性和比例。

**Q4: 装甲板的弧形很难手动建造。**
A4: 使用WorldEdit：
    - 先建一个完整的圆柱壳（外半径R、内半径R-厚度、高度H）
    - 用//sel cuboid选择不需要的弧段并//set air清除
    - 剩余部分即为弧形板
    - 再手动添加破损效果

**Q5: 建筑高度超过了世界高度限制。**
A5: 选项一：使用数据包/mod扩展世界高度上限。
    选项二：将整体下移——将地面基准设为 Y = −36（1.18+支持），
            则结构底部 Y = −36−20 = −56，顶部 Y = −36+378 = 342 < 384。
    选项三：在构建阶段使用1.18+世界（上限Y=320），
            结构从Y=−56到Y=322正好卡在限制内（需略微压缩最高点）。
    选项四：CubicChunks mod（完全解除高度限制）。

**Q6: 紫色锁链的悬垂弧线很难控制形状。**
A6: 先用羊毛沿理想路径放置临时方块确认弧形，
    然后用对应材质替换。悬链线的数学形式为 y = a·cosh(x/a)，
    但实际操作中"看起来自然"即可——不必严格遵循公式。
    
**Q7: 性能问题——40万+方块的结构会导致严重卡顿吗？**
A7: 潜在问题和缓解措施：
    - 光照更新：大量光源方块会导致光照计算负担。
      建议在所有建造完成后再统一放置光源（或使用//fast命令跳过光照更新）。
    - 实体渲染：如果使用了大量盔甲架/物品展示框，会影响帧率。
      本设计尽量避免实体装饰。
    - 区块加载：17×17区块 = 289个区块同时加载。
      在视距10（21×21区块）下大部分结构可见。
      视距16+可以看到完整的外围环境区。
    - 推荐配置：使用Fabric+Sodium优化客户端，建造过程中关闭Shader。

### 15.1 红石驱动的"以太呼吸"系统

**目标**：巨剑金色段纹路节点周期性明暗交替，模拟以太呼吸（约4.8秒周期）。
实现方案：

在巨剑金色段(h=80–150)的纹路交汇节点处，
将原本的荧石替换为红石灯。约需 10–16个节点。

在血槽内部空腔铺设红石线路，连接所有红石灯节点。
血槽在v1.1版中约4格宽×2格高，空间充足。

漏斗时钟（放置于剑身内部）：
两个漏斗面对面相连，放入12个物品。
单程传递≈4.8秒→半周期脉冲≈4.8秒。
从比较器取信号输出到红石线路。

漏斗时钟截面（嵌入血槽内部）：
H → H H = 漏斗（面对面）
↑ → = 物品传递方向
C C = 比较器（从一侧漏斗读取信号）
↓
红石线 → 各节点红石灯

效果：所有金色纹路节点同步亮灭，
周期约9.6秒（亮4.8秒+暗4.8秒），
从远处看巨剑金色段在缓慢"呼吸"。

### 15.2 注入器脉冲传播动画

**目标**：注入器上的脉冲节点依次闪亮，模拟寒极脉冲从剑柄向剑尖传播。
实现方案：

将注入器的脉冲节点（每把8–10个）处的海晶灯替换为红石灯。

在注入器内部铺设中继器(Repeater)链：
每两个节点之间放置2刻延迟的中继器。

节点排列（以注入器#0, 76格长为例）：
剑柄(外端) → N1 → N2 → N3 → ... → N10 → 剑尖(冲击点)

N1到N2之间：约7格距离，用红石粉+中继器连接。
每段延迟2刻→10个节点总传播≈1.8秒。
最后一个节点到循环重启处插入20刻冷却延迟。
完整周期≈3.8秒。

效果：脉冲从剑柄末端开始，
沿注入器轴线向剑尖方向依次点亮每个节点，
如同一颗光子沿剑身飞驰。

空间限制：
注入器截面约11×5格（最宽处）。
红石线路需隐藏在截面内——利用中心2格宽的"血槽"空腔。
对角方向注入器的红石走线会因锯齿问题更复杂。
建议优先在正交方向的4把注入器(#0, #2, #4, #6)上实现。

### 15.3 浑天仪共振闪光

**目标**：浑天仪铭文节点同步闪烁，模拟计算脉动。
实现方案：

在外层环的12–13组铭文位置各放1个红石灯（替代/增加在紫水晶簇旁）。
浑天仪核心球内放置红石时钟（周期10刻≈1秒）。
从核心向外沿外层环体内部走红石线路至各铭文位置。
外层环截面6×6格，中间4×4格空间可走红石线。
所有铭文节点同步闪烁——从远处看浑天仪外环如同呼吸般明灭。
扩展：
中层和内层环也可添加红石灯，但线路空间更紧张（截面仅5×5和4×4）。
建议中内层仅做静态照明，仅外层做动态闪烁。
